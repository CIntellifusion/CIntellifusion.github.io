# Cmake Note

## Abstract

### makefile 

why it is necessary to use makefile instead of shell to compile c/c++ project?

https://www.zhihu.com/answer/1914452432

In short, when the project has several files and takes a relatively long time to compile, using makefile can help you save time, both typing shell commands and compile time. 

when we have a make file, we just need to type ‘make’ in shell to compile. 

Also, makefile is also used in courses to support labs,ICS for example.

```shell
linux make
```



### what is cmake

In general, cmake can generate a makefile automatically to compile the project. 

### why cmake 

Apart from those advantages of makefile, Cmake is able to compile projects across different platform using CMakiList.txt. 

### Xmake 

a more automatically tool to compile projects. 

```shell
linux>xmake 
linux> xmake run
```
